# mini-radas

## Installation

We require Python 3.9 (inclusive) to 3.12 (inclusive). If using `conda`, create a new environment with the appropriate Python version, for example:

```bash
conda create -n mini-radas-3.12 python=3.12
conda activate mini-radas-3.12
```

Then, install the package:

```bash
pip install -e .
```
